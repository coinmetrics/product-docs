# User Guide

The User Guide collects practical, task-oriented documentation for the Coin Metrics Python API Client. Use it to learn the core abstractions, follow recommended patterns, browse runnable examples, and resolve common errors.

- [**Core Concepts**](core-concepts.md) – `DataCollection` objects, DataFrame conversion (pandas / polars / lazy), and CSV / JSON / Parquet exports.
- [**Best Practices**](best-practices.md) – Catalog-driven workflows, parallel exports, lazy execution, wildcards, and other recommendations for getting the most out of the client.
- [**Examples**]() – Runnable end-to-end scripts covering the most common workflows.
- [**Troubleshooting**](troubleshooting.md) – Debugging tools, proxy and SSL configuration, and a reference of common HTTP errors.
